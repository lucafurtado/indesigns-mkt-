# Direção Visual Corrigida — Indesigns

> Este documento substitui todas as direções visuais anteriores que usavam paleta quente (areia, cobre, madeira).
> A paleta real da marca é azul-petróleo, teal e greige mineral.

---

## A Correção Fundamental

Toda a direção visual produzida antes da análise do manual estava **errada na paleta e nas referências**.

| O que estava errado | O que é correto |
|---|---|
| Areia, cobre, madeira quente | Petróleo #173a4a, teal #3c6e71, greige #d1d3c4 |
| Luz quente dourada | Luz fria-neutra, azul hora, manhã direcional |
| Norm Architects / Axel Vervoordt | Niemeyer contemporâneo / Isay Weinfeld |
| Conforto escandinavo | Rigor modernista brasiliense |
| Julius Shulman (anos 50 americano) | Marcel Gautherot (modernismo brasileiro) |
| Textura rattan / linho aquecido | Concreto, pedra, vidro, metal |

## Diretrizes para Toda Criação Visual Futura

### Checklist de aprovação visual

Antes de publicar qualquer conteúdo, verificar:

- [ ] A paleta usa as cores do manual? (#173a4a, #3c6e71, #8ab0ab, #d1d3c4, #ececea)
- [ ] A luz é fria-neutra, não quente-dourada?
- [ ] A tipografia usa KyivType (ou equivalente) + Montserrat?
- [ ] O espaçamento é generoso — não comprimido?
- [ ] A composição tem influência geométrica/modernista?
- [ ] O tom de copy é preciso e contemporâneo — não efusivo?
- [ ] A referência poderia ser de Brasília? (não de Oslo, Milão ou California)
