## SEÇÃO 08 — PROCESSO

**Visual:** 4 etapas em linha horizontal. Número + título + descrição. Linha conectando as etapas.

---

**Título da seção:**
```
Como funciona
```

**Subtítulo:**
```
Do primeiro contato à entrega,
sem surpresas.
```

---

**Etapa 01 — Conversa**
```
Tudo começa com uma conversa gratuita.
Quero entender o seu projeto, o seu espaço e
— principalmente — quem você é.
```

**Etapa 02 — Descoberta**
```
Com o briefing em mãos, inicio o estudo do espaço.
Antes de qualquer render, existe pesquisa, intenção
e um conceito que é só seu.
```

**Etapa 03 — Projeto**
```
Estudo preliminar, renders, escolhas de material.
Você vê o espaço tomar forma antes de qualquer
decisão de obra. Ajustes até estar perfeito.
```

**Etapa 04 — Entrega**
```
Projeto executivo completo para o construtor.
Visitas de acompanhamento durante a obra.
E no dia da entrega, um espaço que finalmente é seu.
```
