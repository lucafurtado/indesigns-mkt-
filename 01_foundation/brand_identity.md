# Identidade de Marca — Indesigns

> Documento atualizado com base na análise profunda do Manual de Identidade Visual 2024.
> Esta é a referência definitiva para toda criação visual e comunicação da marca.

---

## Essência da Marca

**O que a Indesigns entrega:** Arquitetura que traduz quem você é no espaço onde você vive.

**O que a Indesigns representa:** A inteligência do modernismo brasiliense aplicada ao presente. Geometria com alma. Precisão com sensibilidade.

**Como a Indesigns faz sentir:** O cliente sente que chegou a um lugar que o entende — sofisticado o suficiente para confiar, humano o suficiente para se conectar.

---

## O Símbolo

Construído sobre 4 conceitos simultâneos:
1. **A palavra IN** — o início do nome, a ideia de interior, entrada, intenção
2. **O Palácio do Planalto** — a arquitetura de Niemeyer como DNA visual
3. **Brasília vista de cima** — ruas, quarteirões, o Plano Piloto como matriz
4. **Uma porta aberta** — cada projeto é uma entrada em um novo espaço de vida

A letra **E** do logotipo foi modificada para assemelhar-se a uma porta aberta — o detalhe mais autoral da marca, que conecta tipografia e conceito.

## Paleta de Cores Oficial

```
#173a4a  —  Azul-petróleo profundo    (primária / âncora)
#3c6e71  —  Teal médio               (secundária / identidade)
#8ab0ab  —  Teal sage / claro        (terciária / leveza)
#d1d3c4  —  Greige / pedra quente    (neutra / arquitetura)
#ececea  —  Off-white linho          (base / respiração)
```

### Como usar as cores

**#173a4a** — Fundos premium, texto principal em fundos claros, elementos de autoridade. A cor que ancora tudo.

**#3c6e71** — Destaques, CTAs, elementos de identidade, ícones. A cor mais reconhecível da marca.

**#8ab0ab** — Fundos de seções secundárias, elementos gráficos, gradações. Cria leveza sem perder a identidade.

**#d1d3c4** — Fundos alternativos, divisores, elementos de suporte. A cor do concreto e da pedra de Brasília.

**#ececea** — Fundo padrão de conteúdo. Nunca branco puro — sempre este off-white com temperatura.

### O que essa paleta comunica
As cores da Indesigns são extraídas de Brasília:
- O azul-petróleo do céu do planalto ao entardecer
- O teal do Lago Paranoá encontrando o cerrado
- O greige do concreto aparelhado de Niemeyer
- O off-white do linho, do papel, do espaço que respira

**Não é uma paleta de interiores aconchegantes. É uma paleta de arquitetura urbana contemporânea.**

## Sistema Tipográfico

### Títulos — KyivType Sans Medium3
Moderna, clean, elegante. Transmite profissionalismo com criatividade. Permite intervenções e adaptações autorais (como o E-porta). Usar em headlines, títulos de seção, nome da marca.

### Subtítulos — Sweet Sans Pro
Minimalista, contemporânea, fácil leitura. Equilibra o peso dos títulos. Usar em subtítulos, labels, informações complementares.

### Corpo de texto — Montserrat
Minimalista e de alta legibilidade. Usar em parágrafos, textos longos, legendas.

### Diretrizes tipográficas
- Labels e categorias: **VERSALETE ESPAÇADA** (tracking generoso)
- Títulos: tamanho generoso, espaçamento entre linhas amplo
- Nunca comprimir o texto — o espaço é parte do design
- Hierarquia clara: título → subtítulo → corpo, com contraste de peso e tamanho

## Padrões Gráficos

A marca possui 3 padrões derivados do símbolo, todos utilizáveis como elementos de fundo, texturas ou composições editoriais:

**Padrão Azul** — Grid denso de módulos arredondados em #173a4a. Usado para fundos de impacto, capas, elementos gráficos de alta presença.

**Padrão Teal** — Grid arejado em #8ab0ab sobre #ececea. Usado para fundos mais leves, slides, elementos editoriais.

**Padrão Misto** — Composição com arcos, portas e módulos em toda a paleta. Usado para peças mais elaboradas, editorial, apresentações.

## Tom de Voz — Refinado com o Branding

### Atributos do tom (alinhados com a identidade visual)

**Preciso como uma planta baixa** — cada palavra ocupa o lugar certo. Sem excessos.

**Contemporâneo como o teal** — não é vintage, não é nostálgico. Está no presente.

**Enraizado como o azul-petróleo** — tem profundidade, não é superficial.

**Respirável como o off-white** — dá espaço para o leitor. Não sufoca com informação.

**Autoral como o símbolo** — tem ponto de vista. Não poderia ser de qualquer escritório.

### O que o tom NÃO é
- Não é frio nem corporativo (o teal humaniza)
- Não é excitado nem vendedor
- Não é nostálgico nem "clássico"
- Não é genérico de arquitetura

## Referências Estéticas Corretas

Após análise do manual, as referências corretas para a Indesigns são:

**Arquitetura:** Oscar Niemeyer contemporâneo, Lucio Costa, modernismo brasileiro relido pelo presente

**Design editorial:** Publicações como Domus, Casabella, Monacle — rigor visual com conteúdo substancial

**Marcas de referência de identidade:** Studio Mumbai, Kengo Kuma Associates, Isay Weinfeld (pela conexão brasileiro-modernista)

**Paleta visual próxima:** Blu Mediterraneo (a versão brasileira), cerrado como paleta, Brasília como referência cromática

**O que NÃO é referência:** Norm Architects (escandinavo demais), Axel Vervoordt (europeu e antigo demais), Bottega Veneta (luxo italiano sem raiz local)
